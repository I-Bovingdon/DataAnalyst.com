# Assets para la página web

Coloca aquí los recursos que necesita el sitio. Los nombres que el proyecto espera actualmente son:

- `profile.png` → Foto personal (usa este fichero en el `hero` dentro de `avatar-frame`).
- `header.png` → Imagen de cabecera (fondo del header).
- `footer.png` → Imagen de pie de página (fondo del footer).
- `datawave.png` → Imagen decorativa estilo "wave" tecnológica (opcional, puede usarse en hero o secciones).
- `dataflow.png` → Imagen decorativa adicional (opcional).
- `CV_Ismael_Bovingdon_2026_ATS.pdf` → CV en PDF (enlace para descargar desde la web).

Instrucciones rápidas:

- Sitúa todos los archivos anteriores dentro de la carpeta `assets`.
- Si cambias los nombres de fichero, actualiza las referencias en `index.html` e `index-en.html`.
- Para añadir más imágenes o ejemplos de proyectos, sube los ficheros a `assets/` y referencia su ruta desde los HTML.

Ejemplo de uso en `index.html`:

```
<img src="assets/Profile.png" alt="Ismael Bovingdon" />
<a href="assets/CV_Ismael_Bovingdon_2026_ATS.pdf" download>Descargar CV</a>
```

Si necesitas que renombre o mueva archivos automáticamente, dímelo y lo hago.
